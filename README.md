# Création Colis

## Introduction
Création Colis est une application pour gérer les colis. Elle permet de créer, suivre et livrer des colis efficacement.

## Installation
### Prérequis
- Node.js
- Angular CLI

### Instructions
```bash
git clone https://github.com/hichamelhirch/creation-colis.git
cd creation-colis
npm install
ng serve
